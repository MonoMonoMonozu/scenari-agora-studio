# Agora Platz Rakuza
**対話から生まれた知識の広場**

水曜夜の座談会で出てきたトピックを蓄積・分類・検索するナレッジアーカイブアプリです。  
タイトルをペーストするだけで自動分類され、すぐに使える話題を素早く見つけられます。

---

## 使い方

### 1. 入室
ページを開くとパスワード入力画面が表示されます。  
パスワードはメンバーに別途共有しています。セッション中（タブを閉じるまで）は再入力不要です。

### 2. ナレッジを追加する
左パネルの入力欄に、座談会のナレッジページのテキストをそのままペーストします。

**貼り付けテキストの形式（必須）**
```
2026年4月30日 座談会ナレッジ
1. トピックタイトルA
2. トピックタイトルB
3. トピックタイトルC
```

- **日付**：`○年○月○日` の形式が必要です
- **タイトル**：`1.` `2.` など番号で始まる行を自動認識します
- 形式が合わない場合は赤いエラーメッセージが表示されます

解析プレビューで内容を確認してから「▶ テーブルに追加」を押してください。

### 3. Hidden Code を調整する
追加されたレコードのHidden Codeは自動分類されますが、手動で変更できます。

| コード | 意味 | 使いどころ |
|--------|------|-----------|
| 🏛️ 定番 | 普遍的・繰り返し使える知恵 | いつでも引用できる鉄板ネタ |
| 🔥 今が旬 | 今だから刺さるトレンド情報 | 旬のうちに活用する |
| 📦 原石 | 未整理だが可能性あり | 磨けば光るかもしれない素材 |
| ⭐ 授業向き | 教育・講義で即使える | 授業ネタとして即戦力 |

### 4. 検索・フィルター
- 上部のフィルターボタンでHidden Code別に絞り込めます
- 右上の検索ボックスでタイトルのキーワード検索ができます

### 5. 日付の修正
収録日の数字部分をクリックすると編集できます（`YYYY-MM-DD` 形式）。  
Enterで確定、Escapeでキャンセルです。

### 6. データの保存とバックアップ
- **自動保存**：追加・削除・変更のたびにブラウザに自動保存されます（LocalStorage）
- **💾 保存ボタン**：JSONファイルとしてダウンロードできます（定期的なバックアップ推奨）
- **↺ ボタン**：ブラウザの保存データを消去し、初期データ（141件）に戻します

> ⚠️ LocalStorageはそのブラウザ・そのデバイスに紐づいています。  
> 別のPC・スマホで開いても追加データは反映されません。  
> **GitHub Pagesの公開版を正として運用し、定期的にHTMLを更新してください。**

## データ更新フロー（HTMLに新データを反映する手順）

新しいトピックをメンバー全員が使える状態にするには、HTMLファイル自体を更新してGitHub Pagesに反映します。

1. GitHub Pagesの本番URLを開き、新しいトピックを追加する
2. 「💾 保存」ボタンでJSONをダウンロードする
3. 最新の `index.html` を GitHub からダウンロードする
4. HTMLファイル内の `let records = [...]` の中身をJSONのデータで置き換える
5. GitHub Desktopでコミット → Push

> 頻度の目安：座談会2〜3回分まとめてpushするのが現実的です。

## パスワードの変更方法

HTMLファイルをテキストエディタで開き、以下の行を探してください：

```javascript
const PW_HASH = '6f1c08b4f77f3e94c5d2a3e7b9f0d821'; // MD5: agora2024
```

新しいパスワードのMD5ハッシュを生成（例：[md5.cz](https://md5.cz)）して差し替え、GitHub Pagesに反映してください。
